# curses_city
 Basic Sim City style game written in C++ using ncurses for ascii graphics 

# controls
 hjkl - arrow keys (vim style) or use normal arrow keys
 r - Residential
 c - commercial
 w - power plant
 i - industrial
 o - road
 d - rail
 p - police
 f - fire station
 s - School
 m - (medical) hospital
 a - park
 w - power plant
 n - nuclear power plant
 x - clear

 See http://cbabuska.com/CursesCity
